# Superheroes Crossbox

Modern single-page marketing site for a premium CrossFit gym, built with Next.js, React, TypeScript, and GSAP.

## Stack

- Next.js App Router
- React 19
- TypeScript
- GSAP for hero motion and reveal timing
- CSS Modules + global design tokens

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Import the repository into Vercel.
2. Keep the default Next.js framework preset.
3. Build command: `npm run build`
4. Output setting: leave as default for Next.js
